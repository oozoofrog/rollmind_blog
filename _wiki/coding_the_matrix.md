# Coding the Matrix

## Chapter 1

### 함수

#### 1.1 집합에 대한 용어와 표기법
- 1 $$\in$$ [1, 2, 3, 4] 와 같이 기호 $$\in$$은 좌측의 값이 우측집합에 속하는 것을 나타낸다.

#### 1.2 카테시안 곱([[Cartesian]] product)
- 두 집합 A와 B의 카테시안 곱은 a $$\in$$ A, b $$\in$$ B의 모든 쌍 (a, b)로 이루어진 집합
- 두 집합의 크기는 두 집합의 요소의 갯수의 곱과 같다.
